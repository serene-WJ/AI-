# AI-life_assistant
这个库中会储存LLM可以读取的视觉、声音、温度湿度、手表传递的心率/运动数据
#这个知识库专供LLM检索使用

- `/AI_Notes/`: 存放提示词（Prompt）优化技巧与大模型应用指南。
- `/Coding/`: 存放 Python 及前端开发的高频排错笔记。


#algorithm_service:实现姿态数据清洗、关键点平滑、丢帧补偿、左右侧选择、异常帧剔除、关节角度计算、深蹲阶段识别、状态机计数、动作质量评分、心率安全判断、训练容量判断。

#shcemas:增加姿态输入、清洗结果、角度结果、质量评分、心率安全、训练容量、训练报告等数据结构。
兼容两种输入：
1.
{
  "left_shoulder": [320, 120],
  "left_hip": [300, 260],
  "left_knee": [310, 410],
  "left_ankle": [315, 560],
  "confidence": 0.92
}
2.
{
  "keypoints": {
    "left_shoulder": [320, 120, 0.92]
  }
}
